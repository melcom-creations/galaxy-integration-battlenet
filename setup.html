Battle.net Plugin – One-Time Setup

GOG Galaxy 2.0 · Blizzard OAuth Configuration

⚠
Mandatory one-time setup. The shared CLIENT_ID and CLIENT_SECRET used by the community plugin have been revoked by Blizzard. You must register your own free OAuth client before the plugin can authenticate.
1
Open the Blizzard Client Registration
Open this URL in your regular browser (Firefox, Chrome, Edge) and sign in with your Battle.net account. The link takes you directly to the client creation form:
https://develop.battle.net/access/clients/create 📋 Copy Link
2
Fill in the form
Use these exact values:
Client Name GOG Galaxy Plugin - MyClient123  (must be unique, see note below)
Redirect URLs http://friendsofgalaxy.com
Service URL Check "I do not have a service URL for this client"
Intended Use Personal GOG Galaxy 2.0 desktop client plugin to display owned Blizzard games and launch them via the Battle.net launcher. Only used locally on my own PC.
⚠
Client Name must be unique. It has to be unique across all Blizzard developer accounts. Entering the exact name GOG Galaxy Plugin will usually fail with a 500 Internal Server Error, because that name has already been taken by other users of this and similar plugins. Add any distinguishing text, for example GOG Galaxy Plugin - MyClient123, and the form will save immediately.
3
Copy your credentials
Click Save. You will be redirected to Manage Your Clients — the credentials are not shown immediately. To find them:
Step A On Manage Your Clients, click the entry with the name you just chose to open it.
Step B Under Manage Client → Credentials you will see your Client ID displayed directly.
Step C Click Show Secret to reveal your Client Secret. Copy both values.
4
Enter them in consts.py
The file is located in the plugin folder. You can paste the path below directly into the Windows Explorer address bar to get there:
%LOCALAPPDATA%\GOG.com\Galaxy\plugins\installed\battlenet_ba170431-0649-482f-863b-d248592f1842\ 📋 Copy Path
Open consts.py with any plain text editor (Notepad works fine) and replace these two lines:
CLIENT_ID = "your_client_id_here"
CLIENT_SECRET = "your_client_secret_here"
5
Restart GOG Galaxy
Close GOG Galaxy completely and reopen it. The plugin reads consts.py only on startup — changes do not take effect until you restart.
6
Connect in GOG Galaxy
Go to Settings → Integrations → Battle.net and click Connect. You will be prompted to log in with your Battle.net account.
⛔ Keep your CLIENT_SECRET private. Never share it publicly or commit it to a public repository. Anyone with this key can make API calls on your behalf.
